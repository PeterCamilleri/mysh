Help: mysh vls command summary:

The vls command is used to provide version information for the loaded ruby
modules. The usage template for this command is:

mysh>vls <mask>

Where:
 <mask> is an optional, case-sensitive pattern for filtering out the modules
        displayed. If omitted, all modules are displayed.

Examples:

mysh>vls
DidYouMean   1.0.0
Gem          2.5.2
InArray      0.1.8
MiniReadline 0.8.0
Mysh         0.5.4
VersionLS    0.4.1

mysh>vls M
DidYouMean   1.0.0
MiniReadline 0.8.0
Mysh         0.5.4
